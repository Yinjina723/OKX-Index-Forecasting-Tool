# OKX-Index-Forecasting-Tool
这是我个人制作的一个帮助预测指数的工具简单说下流程，我通过OKX客户端上面的历史数据下载到本地，再通过OKX的API获取近三天的K线数据。将两个数据整合再一起，让deepseek自己分析，给出建议和点位，大家自己可以调整各自的需求。我这个对环境要求低，灵活性强。里面有说明 ，关键是比较准确，点位基本都可以预测到。喜欢的可以自己下载试试。由于文件很简单 说明都文档注释中。如果觉得好用记得打赏。有不足的地方，多多原谅。我个人测试了几天都挺准的 还没有同时预测多智股票的的功能，后期再说。最后再把他融合成一个程序。 你如果用我这个预测挣钱了， 记得回来打赏就行，给个茶水
